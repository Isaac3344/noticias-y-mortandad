# Noticias y Mortandad - Proyecto de Práctica

Archivos incluidos: index.html, noticias.html, mortandad.html, css/styles.css, js/script.js, and assets/img/sample.png.

Pasos para subir a GitHub:
1. Crea un nuevo repositorio en GitHub.
2. Inicializa git en la carpeta local y sube los archivos. (git init, git add ., git commit, git remote add origin ..., git push)
3. Opcional: habilita GitHub Pages para publicar la versión estática.
